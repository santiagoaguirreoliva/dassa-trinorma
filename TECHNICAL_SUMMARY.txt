================================================================================
TRINORMA MANAGER — SISTEMA DE GESTIÓN INTEGRADO (SGI) DASSA
ISO 9001 | ISO 14001 | ISO 45001
================================================================================

PROYECTO COMPLETADO: Sistema web full-stack de gestión para la certificación
integrada de Calidad (ISO 9001), Ambiente (ISO 14001) y Seguridad y Salud 
(ISO 45001) de DASSA.

================================================================================
CARACTERÍSTICAS TÉCNICAS
================================================================================

FRONTEND
--------
✓ React 18 + TypeScript
✓ Vite (bundler ultrarrápido)
✓ Tailwind CSS (estilos)
✓ React Router (navegación)
✓ Lucide Icons (iconografía)
✓ Interfaz completamente en español argentino
✓ Diseño responsivo (desktop y tablet)
✓ Sidebar de navegación intuitivo
✓ Autenticación con JWT
✓ Context API para estado global

COMPONENTES REUTILIZABLES
✓ Button, Input, Card, Table, Badge, Dialog
✓ Sidebar con navegación contextual
✓ Layout con topbar
✓ Formularios con validación

BACKEND
-------
✓ Express.js 4.18
✓ SQLite (better-sqlite3) - base de datos embebida
✓ JWT para autenticación
✓ bcryptjs para hash de contraseñas
✓ CORS habilitado
✓ 8 rutas API principales
✓ Middleware de autenticación
✓ Error handling centralizado

BASE DE DATOS
-------------
✓ SQLite local (trinorma.db)
✓ 20 tablas creadas automáticamente
✓ Relaciones con foreign keys
✓ Seed data precargado:
  - 3 usuarios (admin, usuario, auditor)
  - 6 documentos ISO
  - 5 incidentes
  - 8 aspectos ambientales
  - 5 encuestas de satisfacción
  - 10 empleados
  - 5 capacitaciones

AUTENTICACIÓN
-------------
✓ Login con email/contraseña
✓ JWT tokens (24h expiration)
✓ Roles: admin, usuario, auditor
✓ Middleware de verificación
✓ LocalStorage para persistencia
✓ Protección de rutas
